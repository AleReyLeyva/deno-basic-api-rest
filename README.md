# deno-rest-api

### Basic Rest Api with Oak and Deno

### Database: MySQL

### Routes

* GET   -    Get All Users   -  **/users**
* GET   -    Get User      -    **/user/:id**
* POST   -   Create User    -   **/user**
* PUT   -    Update User    -   **/user/:id**
* DELETE  -  Delete User   -    **/user/:id**
